# LeetCode Solutions (C#)

Welcome to my LeetCode solutions repository! 👋

This repository contains my solutions to LeetCode problems implemented in **C#**. The goal is to build strong problem-solving skills, improve DSA concepts, and maintain a well-organized collection of optimized solutions for future reference.

## 📂 Repository Structure

```
LeetCode-Solutions
│
├── Easy
├── Medium
└── Hard
```

Problems are categorized based on their difficulty level.

```
Approach
    ↓

    ↓

```
